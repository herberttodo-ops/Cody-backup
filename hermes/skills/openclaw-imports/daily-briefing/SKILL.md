# Daily Briefing Skill

Quick access to your morning briefing.

## Usage

```bash
# Run briefing now
openclaw run briefing

# Or via python directly
python3 scripts/daily_briefing.py
```

## What It Includes

- **Weather** for Poconos area (Cozie Homes location)
- **Active Lanes** reminder (TLC, Rival, Cozie, full-time role)
- **Quick Hits** checklist tailored to the day of week
- **Daily Focus** quote for motivation

## Cron Schedule

Daily at 8:00 AM EST → Telegram/WhatsApp delivery
